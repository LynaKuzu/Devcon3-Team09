# Project Overview
Players launch a stone using a force-charging mechanic and steer left or right mid-movement to avoid obstacles and aim for high-point zones.

# How to Play

1. Hold the charge key **[Space]** to build power.
2. Release to **[Space]** the stone.
3. Use steering keys  **[A]** and **[D]** to navigate around obstacles.
4. Score updates live as you cross scoring zones or stop in the house.
5. Press **[R]** to reset
